## ⚖️ Comparação entre Java, JavaScript e Ruby

| Critério                 | Java                        | JavaScript                    | Ruby                 |
| ------------------------ | --------------------------- | ----------------------------- | -------------------- |
| **Tipo de linguagem**    | Compilada (bytecode na JVM) | Interpretada (navegador/Node) | Interpretada         |
| **Paradigma**            | Orientada a objetos (forte) | Multiparadigma                | Orientada a objetos  |
| **Tipagem**              | Estática                    | Dinâmica                      | Dinâmica             |
| **Sintaxe**              | Verbosa                     | Média                         | Simples e expressiva |
| **Uso principal**        | Back-end, Android           | Front-end e back-end          | Back-end             |
| **Performance**          | Alta                        | Média/Alta                    | Média                |
| **Curva de aprendizado** | Média/Difícil               | Média                         | Fácil                |
| **Concorrência**         | Threads nativas             | Assíncrono (event loop)       | Limitada             |
| **Uso no mercado**       | Muito alto                  | Altíssimo                     | Menor, mas relevante |
| **Framework famoso**     | Spring                      | React                         | Ruby on Rails        |
